(inner) byFields(fields, params) → {Object}

Builds a filter by fields aggregation stage by checking whether the params value with the same name is truthy or falsy and filters by not null or null respectively in a $match stage.
Parameters:
Name Type Description
fields Array.<String> The fields to filter.

Example
const fields = ['disabledAt', 'createdAt', 'updatedAt'];

const params = {
  disabledAt: false,
};

const stage = filter.byFields(fields, params);

// Stage will be:
{
  $match: {
    disabledAt: {
      $ne: null
    }
  }
}

Example
const ranges = [{
  name: 'yearFrom',
  field: 'year',
  cond: '$gte',
}, {
  name: 'yearTo',
  field: 'year',
  cond: '$lte',
}];

const params = {
  yearFrom: 2012,
  yearTo: 2017,
};

const stage = filter.byNumRange(ranges, params);

// Stage will be:
$match: {
  $and: [{
    year: {
      $gte: 2012
    }
  }, {
    year: {
      $lte: 2017
    }
  }]
}
